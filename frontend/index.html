About Attendora

Attendora automates class attendance using facial recognition. Teachers can start and stop attendance per period with a click. The system stores attendance securely and can notify parents when a student is absent. Designed for fast, accurate, and privacy-aware operation — recognition runs during teacher-initiated sessions only.

  • Period-wise attendance
  • Teacher-controlled start/stop
  • Optional WhatsApp alerts to parents
